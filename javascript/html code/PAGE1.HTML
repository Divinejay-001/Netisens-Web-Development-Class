WEB DEVELOPMENT

Web development refers to the task associated with creating, building and maintaining websites

BRANCHES OF WEB DEVELOPMENT

  1. Front end web development
  2. Back end web development
  3. Full stack web development

TECHNOLOGIES USED IN WEB DEVELOPMENT

HTML: HTML is a standard markup language used for constructing hypertext

Java Script: This is a scripting programming language that allows one to implement complex features on webpages

CSS: Cascading Style Sheet, used for styling and layout

React:It lets you build user interface pieces called components

TOOLS NEEEDED FOR WEB DEVELOPMENT

  1. Internet connection
  2. Text editor
  3. Web browsers
  4. Version control
